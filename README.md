# 01blue.org Landing Page

A premium, sleek digital gateway for 01blue.org, inspired by Hatsune Miku's signature cyans and digital aesthetic.

## Features
- **Aesthetic**: Deep dark glassmorphism with 'Miku Blue' (`#39C5BB`) highlights.
- **Animations**: Driven natively by [Anime.js](https://animejs.com/) for staggered loading entrance and ambient cyber-particle motes in the background.
- **Performant**: 0-build vanilla JS and CSS dependency setup. Runs immediately in any standard browser.

## Links
- **Portfolio**: Routes to the personal portfolio website.
- **Cursorport**: Routes to the custom cursors repository.
